# dark-sky-planner

Astronomy web app.

Search catalogs of deep sky objects and plan your observations with a printable
star chart

## Tools

- Flask

- [D3-celestial](https://github.com/ofrohn/d3-celestial)

- Javascript

## Tutorials

- https://realpython.com/handling-email-confirmation-in-flask/
